# 3-DoF-Robot-Kolunun-leri-ve-Ters-Kinematik-Analizi
Bu proje, 3 Serbestlik Dereceli (3-DOF) düzlemsel bir robot kolunun ileri ve ters kinematik analizi,  ve çalışma alanı (workspace) simülasyonunu içermektedir.
Repository içeriği 3-DoF Robot Kolunun İleri ve Ters Kinematik Analizi.pdf isimli rapor, İleriKinematik.py, TersKinematik.py, Calismaalani.py python dosyaları ile Çalışma Alan.pdf görsellerini içermektedir.
Projede "RRR Planar (Düzlemsel) Kol" konfigürasyonu tercih edilmiştir. 
Kodlar python programlama dili ile oluşturulmuş olup robot parametreleri kullanıcıdan dinamik olarak alınmaktadır.
Çalıştırılabilmesi için  numpy ve  matplotlib.pyplot kütüphanelerinin yüklenmesi gerekmektedir.
